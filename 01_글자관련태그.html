문단을 나누는 태그 : p, pre

문단 영역을 나누는 태그로는 p 태그와 pre태그가 있다 줄바꿈하고자 한다면 별도의 태그필요
그리고 공백은 한 개의 공백만을 표시하기 때문에 별도의      기호문구를 기술해야한다. 

            pre태그는 시작 태그에서부터 종료 태그 까지의 입력된 내용을 그대로 표현
            줄바꿈이라던가      여러개의 공백도 바로 인식함

그 밖에 글자를 다루는 태그들

일반 글꼴 (태그로 감싸지 않은 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더

em : 글자를 기울여서 표시하는 태그

i: 글자를 기울여서 표시하는 태그

mark : 형광펜 효과를 주는 태그

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선 넣어주는 태그

small : 글자를 작게 표현해주는 태그

기본 글자에 sub: 아래첨자 를 나타내는 태그와 sup: 윗첨자를 나타내는 태그

IOTIOT란 각종 사물에 센서와 통신 기능을 내장해 인터넷에 연결하는 기술이다
